#### Evernote9b2
[参考](https://www.alfredforum.com/topic/840-evernote-9-beta-2-for-alfred-3-search-create-append-set-reminders-all-within-alfred/)

| command | description |
|:--:|:---:|
| ent(title) | 按照标题搜索笔记 |
| ens(search)| 根据内容（搜索笔记 |
| enr(reminder) |根据提示搜索 |
| entodo |搜索带有复选框的笔记 |
| enrec | 搜索最近更新的笔记|
| enu | 搜索带有 URL 的笔记|
| enn | 新建笔记 |
| en? | evernote 帮助文档 |
